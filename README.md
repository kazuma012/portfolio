## サイト概要
基本無料で個人が持っている感性に関する自由なテーマで幅広いジャンルの提供を目的としたサイト。
音楽ストリーミング（個人楽曲、コラボ楽曲）だけでなく音楽関連情報（個人の音楽情報やフェス、ライブ会場情報など様々）＋物販販売を一体化したサイト。

### サイトテーマ
有名楽曲だけでなくマイナー曲,mashup,remix,mix,隠し楽曲や世には公表されなかった貴重な曲を配信したい。
そのための個人が持っている貴重な情報を共有するための掲示板、楽曲に関するコメントなどを気軽に共有。
一般ユーザーだけでく個人楽曲で挑戦する若いdj、楽曲制作初心者や音楽関係者にも使ってくれるようなサイト。

### テーマを選んだ理由
自分自身何か作業をする上で必ず音楽の力で物事をこなしてきた経験があるので、モチベーションや作業促進にこのサイトを使ったユーザー全員に
満足してもらいたいから。
間接的ではあるが自分の作ったサイトを利用した若い芽の方々をプロデュースするようなサイトを作るつもりなので、その成長に携われたことに
喜びを感じるから。

### ターゲットユーザ
１０〜３０代をまずはターゲットにして、サイトのテーマでもある幅広い楽曲、ジャンルを追加する上で
ほぼ全ての年齢層が理論上利用可能になっていきたい。

### 主な利用シーン
日常生活、学習中、仕事中、趣味時間など生活する上で音楽を流すことによって
シーンが豊かになる場面
